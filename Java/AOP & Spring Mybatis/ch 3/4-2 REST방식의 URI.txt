
4-2 REST방식의 URI

 : method(CRUD) + uri

게시판과 관련된 이름은 다 /board로 쓰고싶어

메소드 자체에 CRUD방식을 구분함


REST 방식 URI :  : method(CRUD) + uri(parameter)

GET              /board : 	전체게시글 조회
GET              /board?no=3	상세조회
GET              /board/3	상세조회
GET	   /board/post	글등록폼
POST	   /board/post	글등록
PUT              /board/3	수정
DELET          /board/3	삭제
--->근데 board/3 이런건 현재 ajax만 가능함
