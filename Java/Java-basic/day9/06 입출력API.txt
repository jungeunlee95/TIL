
 ★★★★★★ [ 입출력 API ] ★★★★★★
 - ram에 저장된 데이터는 소프트웨어가 종료될 때, 모두 지워짐
 - 실제 컴퓨터에서 실행하는 메인 메모리는 다 ram에 저장
 - but 외부 메모리의 하디스크는 프로그램을 종료해도 정보가 다 남아있음.
->ram에 있는 정보를 잠깐 사라지기 전에 ram에 저장했던 일부분의 데이터를 하디스크에 저장
->다시 소프트웨어 실행하면 하디스트에 있는 정보를 ram에 가져옴
   ==> 입출력 API : 외부장치를 통해서 데이터를 저장 혹은 가져오기 
      -> 램과 외부장치가 데이터를 주거니 받거니 하게 할 수 있는 것 
      -> 다른 언어끼리도 주거니 받거니 할 수 있음, 
      [[-> java.io 패키지에 클래스와 메소드가 다 있음]]


-> 바이트(아스키코드) 기반으로 데이터 주거니   : 바이트가 더 기본임
             or
-> 문자(유니코드) 기반으로 데이터 주거니 받거니

입.출력을 위해서 스트림을 사용함 (byte,character)
[램]            -스트림->           [파일]             -스트림->            [램]
메모리 내용을 파일에 저장(출력)        파일에 있는 내용을 읽어서 메모리로 가져온다(입력)
       OutputStream : Writer                 OutputStream : Reader 
   
1.FIFO : First In First Out
2. 단방향! 출력고ㅏ 입력을 동시에 하지는 않음
