swich문 때에 따라서 n개의 정수중에 선택하고 싶은 만큼 선택해 버릴 수 있음
         - n개의 선택지 중 n개를 선택할수도 있음
       - 다중 선택문이라고도 함
     -자바에서는 스위치를 쓸 때를 정해두고 씀
   -시작을 하고 나면 무조건 스위치케이스의 }-브레이스 표시가 나타날때까지
          모든 입력을 수행함 -> 그래서 1일때 one two tree error 다뜸
    -break; 키워드 사용 가능. -> if에서 쓰면 에러납니다
    
     -스위치에 case에 들어올 수 있는 데이터타입이 한정적임.
    ->1.7 이후부터 : byte, short, char, int, enum, String ->실수형불가능

-> == 동등문이 들어가는 코드는 스위치로 쓸 수 있음(변수에 들어있는 숫자가
뭐인지 궁금할때)
