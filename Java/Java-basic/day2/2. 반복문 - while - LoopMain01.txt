반복문 - 어떤 임의의 문장을 n번 반복할때
while, for, do while

   [필요조건] 반복할 문장 필요
              시작값 필요 ( 이제 몇번째야)
              한바퀴 돌때마다 시작값을 변경하는 코드
              종료조건 
    -> 4가지가 필수 조건은 아님. 일반적으로 필요함


[[[[[   WHILE 형태   ]]]]]]
  시작값;
  while(종료조건) {
        반복할 문장;
        증가/감소;
        }
  sysout("문장");

->순서 시작값-> 조건이 참-> 문장반복 -> 코드 증가/감소 
    ->다시 while -> 참 -> 문장반복 -> 코드 증가/감소
    ->           -> 거짓-> 문장   while문이 거짓일때까지 반복함

무한루프
while(true) {
     프린트
    }    -> 무한루프
