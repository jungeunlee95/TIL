그리고 i를 누르면 그자리에서 insert 모드

o를 누르면 다음줄에 insert 모드가 된다

그리고 끌때는 :wq여기서 w는 저장 q는 종료

등록된 환경변수를 쓰기 위해서 

source  ~/.bashrc 입력

확인하기 위해서

echo $등록한 환경변수이름
