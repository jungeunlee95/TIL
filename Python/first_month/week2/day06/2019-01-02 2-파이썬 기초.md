- **python map**

```python
# n = input()
# nums = n.split()

# a = int(nums[0])
# b = int(nums[1])

# print(a + b)

# 1
# map(내가 적용할 함수, "리스트, 튜플, 딕셔너리")
a,b = map(int, input().split())
print(a+b)

# 2
a,b = [int(x) for x in input().split()]
print(a+b)

# 3
numList = ["1", "2", "3", "4"]
print(list(map(int, numList)))   # -> [1, 2, 3, 4]
```
