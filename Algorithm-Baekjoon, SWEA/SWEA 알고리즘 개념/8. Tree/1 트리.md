**==> 자식으로 가는 것만 차수로 따짐**

방향성 X : 연결된 간선 수

ex) 1-2-3 = 1-1차수 2-2차수 3-1차수

방향성 O : 진입, 진출 차수

ex) 1->2->3 = 1-0,1차수 2-1,1차수 3-1,0차수

트리 : 자식으로 가는 간선만

ex) 1-2-3 : 1-1차수, 2-2차수, 3-0차수
