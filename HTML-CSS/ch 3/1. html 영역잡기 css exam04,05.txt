position : 각각의div를 어느위치에 배치시킬것인가?

-relative : div끼리 서로 조화롭게 움직이겠다
                   원래 위치에서부터 left top이 이동하는 것
                  

-absolute : left, top에 맞춰서 표현(자기의 길을 감)
                  :부모 div에 맞춰서 left와 top값이 맞춰짐
                    서로 덮어쓰기하는 위치에 있을 수 있음

-static : 흐름에 맞춰div를 배치시키겠다 : top값을 고쳐도 영향이 X 
               순서에 맞춰서 표현해주기때문에 , 위치값이 허용되지 않아요
                나오는 순서대로 애들을 나열해주기만함!!
             -position속성이 붙지 않을 때 가지는 디폴트 값

- fixed : 스크롤로 움직여도 움직여 지지 않는 거,
              메뉴바 같이 -> 화면이바껴도 계속 그 위치에 고정
