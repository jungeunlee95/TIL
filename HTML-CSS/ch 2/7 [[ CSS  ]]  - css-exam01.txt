이 속성:값; 을 태그명에 적용해랏
<style type="text/css">
	태그명{ 
	    속성:값;
	    속성:값;
	    속성:값;
	}
                같은 태그명 속성값 또 적으면 그 다음에 나온게 덮어져서 적용됨
</style>

만약 <body>태그명에 인라인css있으면 이게 나중에 적용됨
