## 정렬

```html
<template v-slot:cell(sampleLists)="data">
    <span v-for="list in orderByName(data.item.sampleLists)" v-bind:key="list.id">
        {{list.id}}-{{list.name}}<br>
    </span>
</template>
```

```js
import _ from 'lodash'

methods: {
    orderByPort (a) {
      return _.orderBy(a, 'name')
    }
}
```



## 중복제거

```js
_.uniqBy([{ 'x': 1 }, { 'x': 2 }, { 'x': 1 }], 'x');

// => [{ 'x': 1 }, { 'x': 2 }]
```
